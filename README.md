# Modern Python Calculator 📱

A sleek, functional, and attractive calculator built using Python and the Tkinter library. This project marks the beginning of my journey into GUI (Graphical User Interface) development and event-driven programming.


## 🌟 Features
- **Modern UI:** Dark mode theme with a flat design and professional color palette.
- **Dynamic Calculation:** Supports addition, subtraction, multiplication, and division.
- **Error Handling:** Includes basic "Try/Except" logic to handle invalid mathematical inputs without crashing.
- **Real-time Feedback:** Uses `activebackground` to provide visual cues when buttons are pressed.

## 🛠️ Technical Breakdown
This project allowed me to implement several core programming concepts:
- **Tkinter Widgets:** Practical use of `Tk`, `Button`, and `Entry` widgets.
- **Lambda Functions:** Used for delayed execution of functions, allowing arguments to be passed only when a button is clicked.
- **String Evaluation:** Used Python's built-in `eval()` function to parse and solve mathematical strings.
- **Geometry Management:** Precise placement of elements using the `.place()` coordinate system.

## 🚀 Installation & Usage
To run this project locally, you only need Python installed (Tkinter comes pre-installed with Python).

cd python-calculator
